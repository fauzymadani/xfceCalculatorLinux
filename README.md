
# Calculator Project with GTK



## Version

- GTK#2.0
- Monodevelop 7.8.4 (build 2)
- mono Mono JIT compiler version 6.12.0.200




## Deployment

To compile, use csc:
```bash
   csc MainWindow.cs
```
The compiler will create “MainWindow.exe”, which you can run using:
```bash
   mono MainWindow.exe
```
